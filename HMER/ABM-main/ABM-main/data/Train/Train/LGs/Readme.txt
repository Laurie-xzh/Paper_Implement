Please do not distribute this unofficial data set. An official version will soon be available on the CROHME website and/or on the TC11's data set page.

Content: 

Test2012_LG/Test2012LG/ and Test2013_LG/Test2013LG/: label graph version of the test files for 2012 and 2013 dataset, using inherited edges (so the graphs are DAG).
Test2012_LG/Test2012LG_TREE/ and Test2013_LG/Test2013LG_TREE/: label graph version of the test files for 2012 and 2013 data set, without inherited edges (so the graphs are trees).
